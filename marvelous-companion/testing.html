Testing

Marvelous Companion

How to Join

Send a DM with your preferred email (Google account for Android, Apple ID for iOS) to the developer on Reddit (u/ddaypunk06) or Discord (ddaypunk). You'll receive an invite link to the internal test track on Google Play, or a TestFlight invite for iOS.

What to Test

The current build includes a searchable glossary of game content. Focus on:

  • Navigating between screens — check for crashes or screens that fail to load
  • Browsing and searching Game Sets, Heroes, Villains, Figures, and Components
  • Spelling and grammatical errors in any displayed text
  • Blatantly incorrect game phrasing or content

Known Data Issues

  • The source dataset contained various typos — many have been corrected but some may remain
  • 50% of villain deck data was entered manually; phrasing is generally consistent but double-check anything that looks off
  • Icon placeholders appear in some deck text — this is expected and will be addressed in a future update

How to Update

Keep your invite link handy — tapping it again will install the latest build when a new version is available. Check the changelog to see what's new.

Submitting Feedback

Use the feedback button on the About screen (tap the ⓘ icon in the upper right). It will copy your version number to the clipboard automatically and open the feedback form. You can also email support@punkforge.dev directly.
